# v3 publication positioning

Suggested phrasing:

> AMP-JEPA-Hybrid v3 is an ESM-ready, VAE-based predictive-generative architecture for antimicrobial peptide design that integrates JEPA-inspired masked-view latent consistency, physicochemical regularization, novelty filtering, and external oracle comparison.

Avoid claiming:

> v3 experimentally beats APEX.

Until wet-lab or shared benchmark validation is available, say:

> v3 generates computationally promising AMP candidates that can be prioritized against APEX-style oracle predictions and design constraints.
