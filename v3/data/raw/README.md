# v3 raw data

Place your curated peptide input here before running `v3/run_v3_hybrid.sh`.

Default expected file:

```text
v3/data/raw/peptides.fasta
```

The v3 corpus-preparation script also accepts CSV/TSV files with one of these sequence columns:

```text
sequence
Sequence
PeptideSequence
peptide_sequence
aa_sequence
```
